# S.T.B. OS 
(Simple Terminal Based Operating System)

S.T.B. OS is a 32_Bit OS written in Rust.

The goal of this project is to
make an os similar to msdos and to run a program.

## Features
*Multiple Colors

*Keyboard Support

*Double Fault System

*Async/Await

*Heap Allocation

*"Shortcuts"
